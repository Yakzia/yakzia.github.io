¿Cómo mostrar los archivos ocultos en MacOS?

#Técnico — sábado, 25 de julio de 2020

Hace unos días, mientras aprendía a crear un Log en Electron, me vi en la necesidad de ir a una carpeta del sistema operativo. Para mi sorpresa, la carpeta sí existía, pero no estaba visible.

En internet encontré una forma para poder mostrar las carpetas ocultas del sistema en MacOS a través de la Terminal. Dentro de ella, escribimos la siguiente línea y presionamos Enter

defaults write com.apple.Finder AppleShowAllFiles TRUE && killall Finder

Esta línea le indica al sistema operativo que la aplicación Finder (usada para navegar entre los archivos y carpetas) debe mostrar todos los archivos. Además, deben “matarse” todas las instancias de Finder, esto, para que los cambios tengan efecto.

Para deshacer los cambios, podemos optar por ejecutar una línea similar a la anterior, pero cambiando la palabra TRUE por FALSE

defaults write com.apple.Finder AppleShowAllFiles FALSE && killall Finder
